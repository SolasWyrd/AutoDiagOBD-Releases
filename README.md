# AutoDiag OBD Releases

Публичный канал персональных релизов AutoDiag OBD.

Здесь публикуются только:

- подписанные APK;
- SHA-256 checksums;
- release notes;
- `update.json`, который использует встроенная проверка обновлений.

Исходный код приложения хранится отдельно и не публикуется в этом репозитории.
